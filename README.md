## **Getting Started**

### **Prerequisites**

- Node.js (v14+)
- npm or Yarn

### **Installation**

1. Clone the repository

   ```bash
   git clone https://github.com/your-username/your-project-name.git
   ```

2. Navigate to the project directory

   ```bash
   cd your-project-name
   ```

3. Install dependencies

   ```bash
   npm install
   # or
   yarn install
   ```

### **Development**

To start the development server, run:

```bash
npm run dev
# or
yarn dev

```
